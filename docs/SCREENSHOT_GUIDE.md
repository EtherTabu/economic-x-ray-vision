## Refresh Checklist

- Use the local app running from `npm run dev`.
- Keep the browser width wide enough to show dashboard panels clearly.
- Capture real rendered panels, not mockups.
- Avoid browser extensions, private tabs, or unrelated desktop content.
- If screenshot filenames change, update `README.md` and this guide together.
- Do not add Markdown image links before the referenced image files exist.
