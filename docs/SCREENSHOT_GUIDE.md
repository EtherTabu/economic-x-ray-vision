## Capture Checklist

- Use the local app running from `npm run dev`.
- Keep the browser width wide enough to show dashboard panels clearly.
- Avoid showing browser extensions, private tabs, or unrelated desktop content.
- Capture real rendered panels, not mockups.
- Update README screenshot placeholders only after image files exist.
- Do not add broken image links.

## Suggested Views

- Hero / overview metrics
- Dataset Health panel
- Evidence Dossier Workflow panel
- Intervention Simulator panel
- Archetype Intelligence panel
- Constraint List + Filters
- Expanded card inspection with evidence, action, graph, and archetype sections
