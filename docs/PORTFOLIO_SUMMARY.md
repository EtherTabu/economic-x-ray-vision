# Portfolio Summary

## What It Is

Economic X-Ray Vision is a local-first constraint intelligence engine. It maps hidden friction, scores priority and validation confidence, identifies recurring bottleneck archetypes, separates source provenance from claim support, and turns evidence gaps into validation tasks, evidence packets, and campaign plans.

## Problem It Targets

Many systems underperform because of constraints that are visible to operators but poorly represented in traditional metrics. These include queues, documentation drag, handoff leakage, manual verification, permitting delay, equipment lead time, data fragmentation, and capacity mismatch.

The project models those constraints directly instead of waiting for late-stage outcomes.

## What Was Built

- A Next.js dashboard for browsing and inspecting constraint intelligence records.
- A typed constraint data model.
- A local JSON intake contract and generated app data.
- Deterministic scoring for priority, validation, graph position, and strategy.
- Evidence dossiers and validation workflow classification.
- Source registry and evidence pack generation.
- Validation task, triage, evidence packet, and campaign planning workflow.
- Intervention strategy generation with first experiments and action confidence.
- A reusable constraint archetype taxonomy.
- Cross-industry analog detection.
- Constraint network, comparison, source, validation, campaign, and investigation workspaces.
- Local JSON exports, SQLite artifact generation, and parity audits.

## Technical Stack

- TypeScript
- Next.js App Router
- React
- Local JSON and TypeScript data
- Generated local SQLite artifact
- Node-based local scripts

## Why It Is Credible

The system avoids fake certainty. It does not invent citations, scrape sources, call external AI APIs, or claim unverified ROI. Records are treated as hypotheses until validation improves. Scores are deterministic and inspectable.

The project demonstrates architecture, product thinking, data modeling, local pipeline design, evidence discipline, and the ability to turn an abstract idea into a working analytical system.
